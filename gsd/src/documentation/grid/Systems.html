4 Systems

The systems available are:

  • grid
    Definition of Operations and grid structured data that is completely represented by Common Lisp data structures.
  • grid-tests
    A regression test suite for grid which uses lisp-unit.
  • foreign-array
    Use C (row-major) arrays in foreign libraries from Lisp; requires CFFI.
  • grid-iterate-extension
    Use the iterate system for iterating over parts of grids; requires asdf-system-connnections.
